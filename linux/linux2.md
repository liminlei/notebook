## 1、Vim编辑器
#### 命令模式
```
vim 文件名		   //打开
vim + 文件名		 //打开，定位到最后一行
vim +3 文件名	   //打开，定位到第3行
vim +/imooc abc    //打开定位到abc中imooc第一次出现的地方 
vim aa bb cc		//创建三个文件并且打开
					//底行输入:n向后切换，:N向前切换
```
#### 底行模式
```
:n : 打开多个文件时向后切换
:N : 向前切换
:15 : 定位到15行
```
